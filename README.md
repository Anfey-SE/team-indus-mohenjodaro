# Mohenjo-daro Heritage Preservation Microsite

Built by Team Indus for the Pakistan Preservation Sprint.

An interactive web page presenting the history, urban planning, 
and preservation risks of Mohenjo-daro — one of the world's earliest 
planned cities, located in Larkana District, Sindh, Pakistan.
